# CC3P has moved

`sssstwee/cc3p-download` is no longer the public download/update repository for CC3P.

The active public project is now:

- Repository: https://github.com/sssstwee/cc3p
- Releases: https://github.com/sssstwee/cc3p/releases
- Latest release: https://github.com/sssstwee/cc3p/releases/latest
- Guide page: https://sssstwee.github.io/cc3p/

This repository is kept only as a redirect/compatibility notice for old links.
